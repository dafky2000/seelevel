Scope + filter badgesMirrors viewpoint's active filter automatically (Active / Sold / Any). Shows "Viewport" or "⬡ Zone drawn" depending on whether a geofence is active.
2
Time rangeFilters the accumulated session data by the selected window (based on list_dt for active, sold_dt for sold). Custom allows a date-range picker up to 1 year back.
3
Metric tabsAvg Price · Volume · DOM · $/sqft · List→Sold%. Selecting a tab changes both the summary stats row and the primary chart. All computed from current session data.
4
Export CSVDownloads the time-bucketed aggregated data (e.g. weekly avg + count). Not raw listings — just the chart data points. Safe for Web Store policy.
