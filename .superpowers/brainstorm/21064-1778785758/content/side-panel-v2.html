Scope tabs: Viewport / Session / ZoneViewport = current bounding box only. Session = all accumulated since open, cleared when viewpoint's filter changes. Zone = polygon-filtered session data with coverage tracking.
2
Zone coverage barTracks which API bounding boxes intersected the polygon. Coverage % = union of intersections ÷ polygon area. Estimated total extrapolated from density. Prompts user to pan to fill gaps.
3
Rolling window + alignmentDaily / Weekly / Monthly buckets. Split toggle: "Today" (trailing from now) vs calendar anchor — "Mon" for weekly (default), "1st" for monthly. Override by clicking the anchor label.
4
Always 1 year · pan/scroll to exploreCharts always render 1 full year. Scroll or pinch to zoom in on a period. Export downloads the currently visible window's aggregated buckets as CSV.
